Hands-on writeup focusing on fundamental network discovery using the industry-standard tool, Nmap.

Objective & Environment

The primary goal was to understand the principles of network discovery and learn the syntax of Nmap for basic enumeration on a controlled virtual machine (VM) network.

  • Tool: Nmap (Network Mapper)
  • Environment: Kali Linux / Virtual Machine Setup
  • Focus: Command line proficiency and reading raw network output.

Key Commands & Analysis

Demonstration and analysis of essential Nmap scanning techniques:

1. Ping Scan (Host Discovery)

Used the `-sn` flag to determine which hosts are live on the network, minimizing intrusion attempts.

nmap -sn 192.168.1.0/24

2. Stealth Scan (SYN Scan)

Used the default SYN scan (`-sS`) to find open ports without completing the full TCP handshake, a basic technique for port enumeration.

nmap -sS [Target IP]
