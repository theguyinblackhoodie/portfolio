Project Overview

Omni-Recon is a robust network reconnaissance tool engineered to address common scanning failures in Windows and Android environments. Unlike traditional scanners that often crash due to driver conflicts or strict firewalls, Omni-Recon utilizes a proprietary "Hybrid Scanning Engine" to ensure 100% device detection reliability.

Technical Innovation

  • Hybrid Scanning Engine: Intelligently attempts a fast Scapy/ARP scan first. If driver issues (Npcap/WinPcap) or firewalls are detected, it automatically fails over to a native System Ping sweep.
  • Fing-Style Recognition: Identifies Device Vendors (Apple, Samsung, etc.) and resolves Hostnames for complete network visibility.
  • Smart Interface Detection: Automatically identifies the active internet interface while ignoring virtual adapters like VMWare or VirtualBox.

Integrated Intelligence (OSINT)

  • DNS Lookup: Instantly resolves domain names to IP addresses.
  • GeoIP: Fetches precise location data, ISP information, and coordinates.
  • Whois: Retrieves domain registration, ownership, and expiry details.
